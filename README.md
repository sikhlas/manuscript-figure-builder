### Manuscript Figure Builder
A lightweight, browser-based tool for composing publication-ready multi-panel figures — no installation, no internet connection required. Built for researchers who need clean, journal-quality figure panels with labelling and legend, exportable in multiple formats.
Developed in the Medical Research Core Facility, KAIMRC, KSAU-HS

#### Features
Flexible grid layouts with row and column spanning
Load any image into any panel — images are never cropped (contain mode by default)
Panel labels (A, B, C or custom) with full font and style control
Optional figure legend below the panels
Export as PNG, JPEG, TIFF, or PDF at any resolution
Runs entirely in your browser — your images never leave your computer

####  Getting Started
No installation needed.
Download manuscript_figure_builder.html
Double-click the file — it opens in Safari, Chrome, or Firefox
Start building your figure


#### How to Use
1. Set Up Your Grid
Open the Grid tab on the left panel.
Choose a quick preset (e.g. 2×2, 3×2) or type custom row and column numbers

2. Configure Cell Spanning (Layout Tab)
Open the Layout tab to merge cells — useful when one image is taller or wider than others.
Each cell shows its position (R1 C1, R1 C2, etc.) with a Row span and Col span selector.

3. Add Images
Click any panel in the preview area
A file browser opens — select an image from your computer

4. Add Panel Labels
Open the Labels tab and label it as you wish

5. Add a Figure Legend
Below the grid preview, check the Figure legend box.

6. Export Your Figure
Open the Export tab.
Choose your format.
Click Export — the file downloads immediately.
